# Jogo de Adivinhação Numérica 🎲

Um jogo divertido em Python onde o jogador tenta adivinhar um número secreto dentro de um intervalo, com diferentes níveis de dificuldade.

---

## 🎮 Como jogar

1. Execute o arquivo `teste.py` (ou o arquivo principal do jogo) no Python.
2. Insira seu **nome** quando solicitado.
3. Escolha um nível de dificuldade:
   - **Fácil:** número de 0 a 100
   - **Médio:** número de 0 a 300
   - **Difícil:** número maior (configuração futura)
4. Tente adivinhar o número secreto dentro do número limitado de tentativas do nível escolhido.
5. Receba dicas se o seu palpite foi maior ou menor que o número secreto.
6. Ganhe quando acertar ou perca se esgotar as tentativas.

---

## ⚙️ Funcionalidades atuais

- Três níveis de dificuldade, cada um com intervalos e tentativas diferentes.
- Registro do **nome do jogador**.
- Controle de tentativas por dificuldade.
- Feedback de dicas para cada palpite.
